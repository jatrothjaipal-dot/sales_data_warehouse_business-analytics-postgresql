# Sales Data Warehouse and Business Analytics using PostgreSQL

This project demonstrates an end-to-end data warehouse and business analytics solution using PostgreSQL.

## Project Architecture

* Bronze Layer → Raw data loading
* Silver Layer → Data cleaning and transformation
* Gold Layer → Business-ready reporting views

## Analytics Included

* Database exploration
* KPI analysis
* Ranking analysis
* Segmentation analysis
* Customer report
* Product report

## Tools Used

* PostgreSQL
* SQL
* GitHub
